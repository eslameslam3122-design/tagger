# 🛠️ نظام تاجر - إدارة المبيعات والصيانة

نظام متكامل لإدارة المبيعات، الصيانة، والمحافظ الإلكترونية.

## 📱 المميزات

- **نقطة البيع (الكاشير)**: إضافة منتجات للعربة، بحث، فواتير
- **المخزن والجرد**: إدارة المنتجات، المجموعات، الجرد
- **الصيانة**: استلام الأجهزة، حالات متعددة، كود خاص، بحث، واتساب
- **المحافظ الإلكترونية**: إدارة المحافظ، الميت اليومي/الشهري، معاملات، أرشيف مع بحث
- **العملاء**: إدارة العملاء وإرسال رسائل جماعية
- **المصروفات**: إضافة وعرض المصروفات
- **اليومية**: عرض المبيعات والمصروفات وصافي الربح
- **الإعدادات**: تسجيل رقم واتساب، تخصيص الرسالة
